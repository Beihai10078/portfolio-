# 个人主页 · Portfolio

一个 WinUI 3 Fluent Design 风格的个人主页。

```
════════════════════════════════════════════════════════════════════

  數字三民主義，天下為公
  Digital Three Principles — World for All

════════════════════════════════════════════════════════════════════
```

## ✨ 特性

- 🎨 **WinUI 3 Fluent Design** 设计风格
- 🌓 深浅主题切换（Light / Dark）
- 🌐 中英文双语界面
- 📱 响应式布局（桌面/平板/手机）
- 🎯 滚动平滑动画
- 📋 内容分区：关于我 / 项目 / 技能 / 联系方式
- ⌨️ 快捷键：`Ctrl+T` 切换主题，`Ctrl+L` 切换语言

## 🚀 预览

直接双击 `index.html` 即可在浏览器中打开预览。

线上访问：**https://beihai10078.github.io/digital3.WinUI3/**

## 📝 如何自定义

| 想要修改的内容 | 位置 |
|---|---|
| **个人名字/欢迎语** | 搜索 `hero.title` 或 `hero.greeting` |
| **个人简介** | 搜索 `about.text` |
| **项目卡片** | 搜索 `<section class="section reveal" id="projects">` |
| **技能标签** | 搜索 `skills-` 相关部分 |
| **联系方式** | 搜索 `contact-` 部分 |
| **图标** | 可以直接使用 emoji 表情（✈️ 🎮 💻 🎨）或替换成图片 |

## 📦 部署到 GitHub Pages

1. 将本目录上传到任意 GitHub 仓库
2. 仓库设置 → Pages → 选择 `main` 分支 / `root` 目录
3. 稍等片刻后访问 `https://beihai10078.github.io/digital3.WinUI3/`

## 🎨 设计风格

本项目采用与 [Pixel Pilot](https://github.com/Beihai10078/Pixel-Pilot-WinUI3) 游戏一致的 WinUI 3 / Fluent Design 设计语言：

- **Mica 材质背景**：半透明、柔和的磨砂玻璃质感
- **Fluent 主色调**：`#0078D4` Windows 蓝
- **卡片布局**：圆角 + 阴影 elevation
- **状态分层**：主色/强调色/辅助色/背景色

Made with ❤ and Fluent Design.
